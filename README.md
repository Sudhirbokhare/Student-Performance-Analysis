#  Student Performance Analysis

## Project Overview

This project analyzes student academic performance using Python, Pandas, Matplotlib, and Seaborn. The objective is to identify factors affecting student grades and visualize performance trends through Exploratory Data Analysis (EDA).

##  Objectives

* Perform Data Cleaning and Preprocessing
* Conduct Exploratory Data Analysis (EDA)
* Visualize Student Performance Trends
* Identify Factors Affecting Academic Scores
* Generate Actionable Insights

##  Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Dataset Features

* Gender
* Race/Ethnicity
* Parental Level of Education
* Lunch Type
* Test Preparation Course
* Math Score
* Reading Score
* Writing Score

## Analysis Performed

* Data Cleaning
* Statistical Summary
* Correlation Analysis
* Gender-wise Performance Analysis
* Test Preparation Impact Analysis
* Lunch Type Impact Analysis
* Parental Education Analysis
* Score Distribution Visualization

## Visualizations

* Count Plot
* Histogram
* Scatter Plot
* Bar Plot
* Correlation Heatmap

## Key Insights

* Students who completed the test preparation course scored higher.
* Reading and writing scores are strongly correlated.
* Students with standard lunch performed better on average.
* Parental education positively influences student performance.

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Sudhirbokhare/Student-Performance-Analysis.git
```

2. Install required libraries

```bash
pip install -r requirements.txt
```

3. Run the project

```bash
python student_analysis.py
```
